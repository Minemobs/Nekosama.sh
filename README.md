# Nekosama.sh
A simple downloader written in bash
(previous git history: https://gist.github.com/Minemobs/f552cc0296438e8362cc70f9ab570cfc)

## Example of usage:
`./nekosama.sh "$(./nekosama-tui.sh)"` or `./nekosama.sh URL`

## Requirements:
 - gum (needed for `nekosama-tui.sh`)
 - curl-impersonator
 - jq
